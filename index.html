Web Object-Oriented Protocol for Software and Automation

Get started on GitHub

Woopsa is a simple, open-source protocol for machine to machine (M2M) communication. It allows you to supervise yours devices and applications by using a simple web browser, or a complete desktop application.

Simple

Woopsa as been designed to be as simple as possible. It is easy to understand, easy to integrate and easy to use!

Open-source

Woopsa is fully open-source. This means a lot of good things, starting with the fact that it will be free, and forever...

Scalable

Woopsa is already available for Microsoft .NET applications, Beckhoff PLC, as well as for those little Arduinos.

RESTful

Woopsa is based on HTTP + JSON and expose your applications and devices as a well-known and flexible REST API. Use your web browser to access them!

Robust

Despite his young age, Woopsa is already very solid thanks to the use of extremely widespread technologies.

Secure

A version using HTTPS and authentification is on the road. Because Woopsa is based on standards, it will instantly benefit from years of experience.
