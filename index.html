oopsa

Web Object-Oriented Protocol for Software and Automation

Get started on GitHub

Simple

Woopsa as been designed to be as simple as possible. It is easy to understand, easy to integrate and easy to use!

Open-source

Woopsa is fully open-source. This means a lot of good things, starting with the fact that it will be free, and forever...

Scalable

Woopsa is already available for Windows desktop application. A version for Beckhoff PLC is already on the road, as well as one for those little Arduinos.

RESTful

Woopsa is based on HTTP + JSON and expose your applications and devices as a well-known and flexible REST API.

Robust

Despite his young age, Woopsa is already very solid thanks to the use of the extremely widespread HTTP protocol.

Secure

A version using HTTPS and authentification is on the road. Because Woopsa is based on standards, it will instantly benefit from years of experiences.
